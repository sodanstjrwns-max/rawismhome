## 10. 리스크 관리 가치

### 10.1 플랫폼 의존도 감소

**네이버 플레이스만 의존할 때의 리스크:**
```
❌ 네이버 알고리즘 변경 → 노출 급감
❌ 악성 리뷰 테러 → 평점 하락
❌ 경쟁자 광고 입찰 → 비용 상승
❌ 플랫폼 정책 변경 → 대응 불가
❌ 네이버 장애 → 예약 불가
```

**자체 웹사이트가 있을 때:**
```
✅ 네이버 의존도 50% → 30%로 감소
✅ 구글 검색 유입 확보 (분산)
✅ 직접 유입 경로 확보
✅ 플랫폼 변화에 덜 취약
✅ 브랜드 자산 보호
```

### 10.2 마케팅 비용 리스크 헷지

**광고비 의존 구조:**
```
매출 = 광고비 × 전환율 × 객단가

문제점:
- 광고비 상승 → 수익성 악화
- 광고 중단 → 매출 급감
- 경쟁 심화 → CPC 상승
```

**SEO + 광고 병행 구조:**
```
매출 = (SEO 유입 + 광고 유입) × 전환율 × 객단가

장점:
- SEO 유입은 고정 비용 0원
- 광고비 효율 측정 가능
- 광고 중단해도 기본 매출 유지
```
